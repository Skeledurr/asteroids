# Asteroids

A small arcade project built with **Unity 6.3 LTS**, inspired by the classic Asteroids game.

> **This repository uses Git LFS**  
> After cloning, make sure you run:
>
> ```bash
> git lfs pull
> ```


### Controls

> Ship Thrust / Reverse - Arrow Up / Down
> 
> Turn Left / Right - Arrow Left / Right
> 
> Shoot - Space
> 
> Slo-Mo - Shift (hold down)


### Slow Motion Mechanic

> The player can activate **Slo-Mo** by holding down Shift. This will slows down only asteroids while active.
> 
> Slo-Mo has energy that will drain while in use. When it reaches 0, Slo-Mo will automatically turn off.
> 
> Slo-Mo energy automatically recharges while Slo-Mo is inactive.
